# Growmodo Theme

Growmodo is a custom WordPress theme for real‑estate agencies. It provides a clean, mobile‑first design with responsive layouts, custom post types (Property, Testimonial, FAQ), and accessibility enhancements such as proper `alt` text and ARIA labels.

## Features
- Responsive header, footer and hero sections
- Custom post types: `property`, `testimonial`, `faq`
- Accessible images with descriptive `alt` attributes
- ARIA labels for navigation, buttons and carousel controls
- Gutenberg‑compatible styling and CSS variables for easy theming
- SEO meta description on the front page

## Installation
1. Clone or download the repository.
2. Place the theme folder inside `wp-content/themes/`.
3. Activate the theme from the WordPress admin.
4. Import the provided demo content if desired.

## Usage
Use the WordPress admin to manage properties, testimonials and FAQs. The front page displays a hero section, feature cards, a slider of featured properties, testimonials, FAQs and a CTA banner.

## Development
Run `npm install` and `npm run dev` if the theme includes a build step (e.g., for assets). Follow the standard WordPress development workflow.
