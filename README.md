# Growmodo Theme

A lightweight, mobile‑first WordPress theme for real‑estate agencies.

## Features
- Responsive layout with header, footer and hero sections
- Custom post types: `property`, `testimonial`, `faq`
- Accessibility: descriptive `alt` attributes, ARIA labels
- Gutenberg‑compatible CSS variables for easy theming
- SEO meta description on front page

## Usage
Manage properties, testimonials and FAQs via the WordPress admin. The front page showcases a hero banner, feature cards, property slider, testimonials, FAQs and a CTA.
